⛵ Sandeq Boat Racing

Game balap perahu berkecepatan tinggi yang dibangun dengan Three.js, terinspirasi oleh kapal layar tradisional Sandeq dari suku Mandar di Sulawesi Barat, Indonesia.

Gambaran Umum

Sandeq Boat Racing menghadirkan sensasi budaya maritim tradisional Indonesia ke layar Anda. Navigasi melalui perairan yang menantang, hindari rintangan, dan bersainglah dengan perahu lawan dalam balapan arkade 3D yang seru ini.

Sandeq diakui sebagai salah satu kapal layar tradisional tercepat di kepulauan Austronesia, melambangkan keahlian pelaut ulung masyarakat Mandar.

Fitur Utama

- Warisan Budaya: Rasakan permainan yang terinspirasi oleh sejarah maritim Sulawesi Barat yang kaya.
- Lingkungan 3D Dinamis: Didukung oleh Three.js dengan latar belakang laut yang terus bergerak dan efek visual yang imersif.
- Rival AI Cerdas: Bersainglah dengan 5 perahu NPC yang memiliki logika penghindaran rintangan yang cerdas.
- Dukungan Multi-Platform: Desain responsif dengan kontrol keyboard untuk Desktop dan Joystick kustom untuk Perangkat Seluler (Mobile).
- Rintangan Menantang: Navigasi melalui terumbu karang, kayu terapung, hingga perahu lainnya!

Cara Bermain

Kontrol

Perangkat | Aksi | Input
:--- | :--- | :---
Desktop | Mengemudi | Tombol Panah atau WASD
Mobile | Mengemudi | Joystick Virtual (Kiri Bawah)
Semua | Jeda (Pause) | Tombol Pause (Kanan Atas)

Tujuan
- Mulai Balapan: Klik START dari menu utama.
- Navigasi: Kemudikan perahu Anda sepanjang lintasan 16.000 unit.
- Hindari: Tabrakan dengan pulau atau rintangan akan memperlambat dan memukul mundur perahu Anda!
- Menangkan: Jadilah yang pertama melewati garis finis!

Instalasi & Pengaturan

Untuk menjalankan game ini secara lokal:

1. Clone repositori:
   git clone https://github.com/NabilQastari/Sandeq-Racing.git

2. Buka proyek:
   - Cukup buka file index.html di browser modern apa pun.
   - Rekomendasi: Gunakan server pengembangan lokal (seperti ekstensi Live Server di VS Code) untuk pengalaman terbaik.

Struktur Proyek

- index.html: Struktur utama game dan antarmuka (UI).
- script.js: Logika permainan, pengaturan scene Three.js, kelas pemain/NPC, dan deteksi tabrakan.
- style.css: Gaya UI modern dan tata letak responsif.
- /assets: Tekstur game, file audio, dan elemen UI.
